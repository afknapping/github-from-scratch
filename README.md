# GitHub from Scratch

    $ bundle install
    $ jekyll serve -w --baseurl ''

(`-w` also watches files in `_posts` folder for updates. giving baseurl necessary because github pages config (baseurl = repo name).)

Then goto <http://localhost:4000/>.
